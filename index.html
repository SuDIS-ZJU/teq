T. E. Q.

An Open and User-friendly Testbed for Edge Computing based Query Processing Algorithms

The use of edge computing as a means of querying dense and rapid data streams has shown great promise, but developing and assessing algorithms for this paradigm are still laborious for researchers. Current simulators are unable to execute real computations and the decentralized nature of edge computing algorithms makes them hard to implement and test. To address these issues, we propose TEQ (Testbed for Edge computing based Query processing algorithms), which is designed to be (1) open, enabling extensible simulation and new algorithm deployment; (2) user-friendly, simplifying currently heavy simulation, complex algorithm module management, and repetitive evaluation. TEQ covers various techniques to achieve the design goals, such as a method to streamline decentralized algorithm modules as one executable Flink script, the data/query playback control to ensure reproducible streams during evaluations, and a means to transform Flink evaluation metrics into desired metrics associated with user-customized edge computing runtimes. TEQ is in the open source process to extract external contributions and advance edge computing based querying techniques.
